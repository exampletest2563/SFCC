Creating Custom Tags with <ismodule>

There are three key ISML files required for creating and using a custom tag:

  1. The ISML file that sets the values of any attributes for the custom tag (e.g., modules.isml).
  2. The template attribute specifies the ISML file that defines what happens when the attributes are passed. For example, see the contents of app_storefront_base/cartridge/templates/default/components/content/contentAsset.isml.
  3. Invoke the custom tag inside an ISML template, such as pageFooter.isml.
