Site Preferences

Site Preferences are another way to store and retrieve data from your sandbox. They are, in fact, one of the System Object Types.

Check Site Preferences Fields

  • Go to Administration > Development > System Objects Types
  • Click on the Site Preferences item
  • Click on the Attribute Definitions tab and review the fields
  • Click on the Attribute Grouping tab and review the defined groups

As you can see, Site Preferences have many fields. Much like you did with the Product System Object, try creating a new field for this object.

Adding Value to SitePreferences Fields

  • Go to Merchant Tools > Site Preferences > Custom Preferences
  • You will see that, like other System Objects, fields are organized according to the Field Groups we created.
  • You can either use one of the existing groups or create your own in the Attribute Grouping tab
  • Click on the desired Grouping (for example, Storefront Configs)
  • You can now view and modify the values stored for each field

Retrieve Site Preferences in Code

To retrieve any Site Preference using code, you only need to know the field’s ID and use the following code:

  • If retrieving a Site Preference created out-of-the-box with your site: 
    dw.system.Site.getCurrent().getPreferences().
  • If retrieving a Custom Site Preference created by you or your team: 
    dw.system.Site.getCurrent().getPreferences().custom.

Exercise: Add a New Field to the Product System Object and Display It

  • Go to Administration > Development > System Objects Types
  • Click on the Product item
  • Click on the Attribute Definitions tab
  • Click New
  • Input the following values:
    • ID: salesforceb2c_certification_questions_com
    • Display: New Attribute SFRA Student Guide
    • Value Type: String
  • Click Apply
  • Click on the Attribute Grouping tab. Create a new Grouping:
    • ID: salesforceb2c_certification_questions_com_group
    • Name: SFRA Student Guide Grouping
  • Click Add
  • If your new group is not displayed, click All at the bottom
  • Click on the number in the same row as your new grouping
  • Under Assign Attribute Definition, click on the ... button
  • In the pop-up, enter part of the ID of your field in the search field and click Find
  • Click the checkbox for your field and click Apply
  • Click Back
  • Now go to any product Edit Page and search for your grouping and field
