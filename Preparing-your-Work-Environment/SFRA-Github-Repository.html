Salesforce maintains the official code for SFRA in the following GitHub repository. This repository serves as a reference for any other repository you might find that contains SFRA code. It is also the place to check for new releases or official bug fixes.

NOTE: To access this repository, you must (more information can be found here):

  • Enable Two-Factor Authentication (2FA) for your GitHub account.
  • With a valid and logged-in GitHub user account (with 2FA configured), associate that account with Commerce Cloud by visiting the Salesforce Commerce Cloud SSO Signup page. This will connect your GitHub user account to your Account Manager user account and establish the SSO relationship.
