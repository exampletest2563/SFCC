Site Path on Business Manager

When we refer to a "path," consider it as a form field listing all cartridges used in the project, separated by a colon. Each site has its own path.

Uploading your code to the Sandbox is a mandatory step for your code to function on your storefront, but it is not the only requirement. To ensure a site uses a cartridge, you must add the cartridge to the site's path in Business Manager.

To do this:

  1. Open Business Manager.
  2. Navigate to Administration > Sites > Manage Sites > RefArch > Settings Tab.
  3. Locate the Cartridges field. This field represents the path we discussed.
  4. Add the new cartridge to the cartridge path, positioning it before the app_storefront_base cartridge. (Remember, if there are duplicate files in different cartridges, the first one found, based on the order in the Cartridges field, will be used.)
  5. Ensure that site caching is disabled for the site.
