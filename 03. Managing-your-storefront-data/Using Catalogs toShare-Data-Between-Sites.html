Using Catalogs to Share Data Between Sites

The RefArch import contains both site-specific data and data shared among all sites in the organization. The RefArch catalogs are available to the empty site you created earlier.

There are two types of catalogs:

  • Master catalogs define all shared products of an organization.
  • Site catalogs specify category navigation and products for a particular site. Therefore, a site can have only one site catalog.

A site can have one or more master catalogs. Because multiple sites can share a master catalog, you can specify some attributes at the site level. For example, using the OnlineFlag attribute enables a product to be online in one site but offline in another.
