Categories

When you begin your storefront site implementation, you will identify the structure and hierarchy for your categories. Categories and products are frequently imported from a legacy system.

Access categories through their respective catalogs. To edit categories in Business Manager:

  1. Go to Merchant Tools → Products and Catalogs → Catalogs → {select a catalog}.
  2. Click the Edit button for a category.

The following screen will display:

It contains the following tabs:

  • General Tab: This tab identifies the general attributes for the category (for example, category ID, name, description, online status, and online dates).
  • Select Language (Locale): Selecting a language enables you to enter text for that language. For instance, if you enter French text, customers must switch their storefront language to French to view that text. This feature is used in categories, products, and elsewhere in Business Manager.
    • Example: If your site supports both English and French, you would enter the category name “Men” in English and then switch to enter “Homme” in French.
  • Category Attributes Tab: This tab enables you to specify Search Engine Optimization, Sitemap Attributes, Presentation Attributes, Search Ranking, and other custom attributes.
  • Product Attribute Definition Tab: This list shows attribute groups attached to this category. Category attribute groups define product attributes for all products bound to this category. Provide an ID and click Add to add a new attribute group to the category.
  • Search Refinement Definitions Tab: This tab identifies the search refinements for each category. This allows customers to refine a search using multiple attributes, such as price, category, or product attributes. This reduces the quantity of products displayed on the storefront.
    • Customers may use search refinements to find TVs based on screen size, price, and manufacturer. This is particularly useful for filtering a large catalog when a customer is looking for very specific products.
    • Customers may use search refinements to reduce the quantity of products displayed so that only women’s shoes in size 4 ½ are shown on the storefront.
  • Page Meta Tag Rules: This tab shows the rules applied for the current category to display meta tag values such as name, description, robots, title, and so on.
  • Categorization Rules: This tab allows you to define rules to add products to this category.
  • Magnifying Glass Icon: This icon enables you to preview the category on the Storefront.
