Introduction

Controllers are server-side scripts written in JavaScript and Salesforce B2C Commerce Script that handle storefront requests. They manage the flow of data in your application, create ViewModels to process each storefront request as a route, and generate an appropriate response.

For example, in a storefront application, clicking a category menu item or entering a search term triggers the controller that renders the page.

Controllers can be broken down into smaller pieces, making it easier to analyze each part individually and understand its responsibilities. We will use the Cart.js controller, located at app_storefront_base/cartridge/controllers/Cart.js, as a reference for this module.
