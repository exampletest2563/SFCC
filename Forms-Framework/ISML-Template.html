ISML Template

Define an ISML template with the necessary tags for a valid HTML form: <form>â€¦</form>

You can implement your own form action by specifying a controller URL, but this would bypass the Forms framework.

<form action="${URLUtils.url('<Controller>-<Function>')}" method="post">
       ...
</form>
  • The method URLUtils.url('<Controller>-<Function>') will call the specified controller and function when the submit button for the form is pressed.
  • Another method, URLUtils.continueURL(), ensures that the form is submitted back to the controller that displayed the form template.
  • SFRA uses the input tag for creating form fields.
